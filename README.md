# Calculator Console

Простой консольный калькулятор для выполнения базовых арифметических операций.

## Описание

Консольный калькулятор для выполнения базовых арифметических операций. Это была первая версия проекта, которая впоследствии развилась в полнофункциональный калькулятор с GUI и поддержкой различных типов данных.

## Функциональность

- Базовые арифметические операции
- Парсинг математических выражений
- Обработка ввода/вывода
- Базовые алгоритмы вычислений

## Технологии

- **C++17** - основной язык программирования
- **Парсинг математических выражений** - обработка пользовательского ввода
- **Обработка ввода/вывода** - работа с консолью

## Структура проекта

```
calculator-console/
├── main.cpp        # Точка входа
├── calculator.h/cpp # Логика калькулятора
└── README.md       # Документация
```

## Требования

- Компилятор с поддержкой C++17

## Сборка

```bash
g++ -std=c++17 -O2 main.cpp calculator.cpp -o calculator
```

## Использование

### Запуск приложения

```bash
./calculator
```

### Пример использования

```
Введите выражение: 2 + 3 * 4
Результат: 14
```

### Поддерживаемые операции

- Сложение (+)
- Вычитание (-)
- Умножение (*)
- Деление (/)

## Эволюция проекта

- **Версия 1 (текущая)**: Простой консольный калькулятор
- **Версия 2**: Добавлен графический интерфейс на Qt (см. Calculator Qt Basic)
- **Версия 3**: Добавлена поддержка множества типов данных и MVC архитектура (см. Calculator Qt)
